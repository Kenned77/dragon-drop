
# Dragon Drop

A tile-matching game.

## View site

[dragon-drop.xyetrvyk25.us-west-2.elasticbeanstalk.com]

## Run locally

Download and install project:
```
git clone git@github.com:bananno/dragon-drop.git
cd dragon-drop
npm install

Start local server:
```
npm start
```

Local address:
```
http://localhost:5000/
```
